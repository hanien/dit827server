Smart Controls


Adjust your car's environment so it's the best for you!

The Purpose

The Smart Controls system was created in order to improve the car experience for all people in the car. It does this by memorizing each persons ideal car condition so that the car can provide those ideal conditions

Profiles

Profiles are the way the system remembers what each person wants in the car. A person can create their own profile and set their preferences in it. The system will then act to get the conditions specificed in the profiles.

Zones

The inside of the car is divided into 3 zones, 1 for the driver seat, 1 for the passenger seat and 1 for the backseat. Each zone can be assigned its own profile and the system will match the profiles preferences with the conditions detected in the zone. Each zone has its own sensors which are used to see the state of each zone.

Device Handler

Using IOT the system can also control devices connected to the car in order to make the driving experience better. For example it can mute or lower the volume of phones if the driver wants to make it quieter.
